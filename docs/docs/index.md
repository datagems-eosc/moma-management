# DataGEMS MoMa Management

This is the documentation site for the DataGEMS MoMa Management Api. The service is part of the wider [DataGEMS](https://datagems.eu/) platform. This is part of the overall [platform documentation](https://datagems-eosc.github.io/).

The MoMa Management service is a middleware that provides access to the MoMa Object Catalog and is responsible for managing the MoMa graph in Neo4j. The service relies on Cypher queries to manage MoMa objects supporting insertion, updates, and deletion within the catalog. It exchanges information with other components using JSON-formatted data.
